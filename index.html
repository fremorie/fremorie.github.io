Physics Memorizer

Physics Memorizer

This is a game I created for my younger brother to help him memorize physics formulas.

The idea is based on the Memory card game (also called "Concentration"): all of the cards are laid face down on a surface and two cards are flipped face up over each turn. The object of the game is to turn over pairs of matching cards.

In this version, each pair represents a formula: one card has a physical quantity on it, the other has a calculation formula for the physical quantity.

Pomodoro Timer

A simple, easy-to-use pomodoro timer.

This website allows a user to set custom time intervals and sends notifications (if allowed) when an interval ends. When a pomodoro is "eaten" (a 25-minute interval of work has ended), a little tomato appears at the bottom of the screen. This allows you to see how many 25-minute sessions of work you have currently done.
